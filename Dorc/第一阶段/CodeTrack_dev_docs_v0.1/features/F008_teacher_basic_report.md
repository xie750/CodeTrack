# F008 教师基础反馈

## 目标

让教师快速知道常见错因、提示依赖和未完成学生，并获得可执行建议。

## 页面

首个 Demo 必做：

- 课程提交列表；
- 单个学生时间线。

首个 Demo 可选展示，但不作为主验收阻塞项：

- 错因统计；
- 提示层级分布；
- 未完成学生；
- 教学建议。

## 规则

- 统计由数据库聚合；
- AI 只根据统计生成建议文案；
- 建议与数据事实分区；
- 小样本时明确样本量；
- 教师可以复核诊断。
- 如果统计或建议使用预置样例数据，页面或演示材料必须标注数据来源。

## 验收

- 教师能查看当前课程学生；
- 能进入单个提交时间线；
- 能看到错因和最高提示层级；
- 时间线包含提交、执行、测试、诊断、提示和能力证据事件；
- 班级统计如进入首个 Demo，则必须与底层记录一致；
- AI 建议如进入首个 Demo，则不得伪造不存在的数据。
